Some times, when I'm doing edits directly, that is, without some agent (yes, I still write code myself!), I do want a LLM-generated, [conventional-commit](https://www.conventionalcommits.org/en/v1.0.0/) message.  

That's usually a single message to claude "give me a conventional commit message for this diff", which later I converted to a skill. This allows me to tune some nuances arround when to use `chore` vs `build`, etc,.  

However, even the use of the skill became a bit tedious, so I wrote a small set of `git` functions for fish shell that I use daily.
